## 基于OpenGL的简单渲染器

- ### 运用imgui进行交互

- ### 实现了摄像机类，shader类，光源类，模型类,渲染模式，填充or线框

  摄像机类运用欧拉角进行实现，光源类有平行光，点光源和聚光，默认只有一个平行光和一个聚光灯，点光源可以添加，最大支持48个点光源，模型类依赖assimp库实现。
